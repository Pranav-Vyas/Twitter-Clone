# Twitter-Clone

Twitter-Clone is an attempt to implement some of twitter's functionalities.

## Features:
- Login/Signup
- Tweet
- Retweet
- Like
- Comment
- Bookmark
- View profile
- View post
- Explore
- Follow
- Sending messages in real time
- Upload profile photo and cover photo, etc. 

## Technologies used: 
- HTML
- CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io

## Packages used:
- material-ui
- socket.io
- bcrypt
- express
- express-session
- jsonwebtoken
- mongoose
- multer
